## Strategy shift from spec

The original spec called for a rebase onto upstream, but the local `2464e99` is a fresh root commit (a snapshot, not a true clone) and the two histories share no common ancestor. The plan was amended to use `git cherry-pick` instead. The end-state of `sync/upstream-2026-06` is equivalent: it contains every commit on `upstream/main` (because the snapshot was the upstream tree at the time) plus the 9 sandbox-only commits applied as if they had been made on top. Per-file differences vs `upstream/main` are exclusively the 47 sandbox-only files (CLAUDE.md, `.claude/skills/`, `docs/superpowers/`, README test docs, Mermaid diagram).
